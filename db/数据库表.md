### 用户表：
+---------------+---------------------+------+---------------+-------------------+----------------------------------+
| 字段名        | 数据类型            | 必填 | 键 / 索引     | 默认值            | 描述说明                         |
+---------------+---------------------+------+---------------+-------------------+----------------------------------+
| id            | bigint(20) unsigned |  是  | PK (主键)     | AUTO_INCREMENT    | 用户唯一标识，自增ID             |
| email         | varchar(255)        |  是  | UNIQUE (唯一) | -                 | 用户邮箱，作为登录账号           |
| password      | varchar(255)        |  是  | -             | -                 | 登录密码（依需求明文存储）       |
| is_subscribed | tinyint(1)          |  是  | -             | 0                 | 是否订阅邮件：0-未订阅，1-已订阅 |
| created_at    | timestamp           |  是  | -             | CURRENT_TIMESTAMP | 账号注册时间                     |
| updated_at    | timestamp           |  是  | -             | CURRENT_TIMESTAMP | 信息最后更新时间                 |
+---------------+---------------------+------+---------------+-------------------+----------------------------------+

### 每日运势表 (fortunes)
+-------------+---------------------+------+---------------+-------------------+----------------------------------+
| 字段名      | 数据类型            | 必填 | 键 / 索引     | 默认值            | 描述说明                         |
+-------------+---------------------+------+---------------+-------------------+----------------------------------+
| id          | bigint(20) unsigned |  是  | PK (主键)     | AUTO_INCREMENT    | 运势记录唯一标识，自增ID         |
| user_id     | bigint(20) unsigned |  是  | UK (联合唯一) | -                 | 关联的用户ID (users.id)          |
| target_date | date                |  是  | UK / IDX      | -                 | 运势所属的日期 (如: 2026-04-19)  |
| content     | text                |  是  | -             | -                 | 大模型生成的今日运势详细内容     |
| created_at  | timestamp           |  是  | -             | CURRENT_TIMESTAMP | 运势数据生成并写入的时间         |
+-------------+---------------------+------+---------------+-------------------+----------------------------------+

### 验证码记录表 (verification_codes)
+---------------+---------------------+------+---------------+-------------------+----------------------------------+
| 字段名        | 数据类型            | 必填 | 键 / 索引     | 默认值            | 描述说明                         |
+---------------+---------------------+------+---------------+-------------------+----------------------------------+
| id            | bigint(20) unsigned |  是  | PK (主键)     | AUTO_INCREMENT    | 记录唯一标识，自增ID             |
| email         | varchar(255)        |  是  | IDX (联合索引)| -                 | 接收验证码的邮箱                 |
| code          | varchar(10)         |  是  | -             | -                 | 发送给用户的具体验证码           |
| business_type | tinyint(4)          |  是  | IDX (联合索引)| -                 | 业务场景标识：1-注册，2-重置密码 |
| expires_at    | timestamp           |  是  | -             | -                 | 验证码过期时间(如发送时间+5分钟) |
| created_at    | timestamp           |  是  | -             | CURRENT_TIMESTAMP | 验证码生成和发送的时间           |
+---------------+---------------------+------+---------------+-------------------+----------------------------------+

### 用户运势资料表 (user_profiles)
+---------------+---------------------+------+----------------+-------------------+----------------------------------+
| 字段名        | 数据类型            | 必填 | 键 / 索引      | 默认值            | 描述说明                         |
+---------------+---------------------+------+----------------+-------------------+----------------------------------+
| id            | bigint(20) unsigned |  是  | PK (主键)      | AUTO_INCREMENT    | 记录唯一标识，自增ID             |
| user_id       | bigint(20) unsigned |  是  | UNIQUE (唯一)  | -                 | 关联用户ID，单用户一份资料       |
| birthday      | date                |  是  | -              | -                 | 出生日期                         |
| constellation | varchar(32)         |  是  | IDX (普通索引) | -                 | 星座                             |
| gender        | varchar(16)         |  是  | -              | -                 | 性别                             |
| city          | varchar(64)         |  是  | -              | -                 | 所在城市                         |
| occupation    | varchar(64)         |  是  | -              | -                 | 职业                             |
| created_at    | timestamp           |  是  | -              | CURRENT_TIMESTAMP | 创建时间                         |
| updated_at    | timestamp           |  是  | -              | CURRENT_TIMESTAMP | 更新时间                         |
+---------------+---------------------+------+----------------+-------------------+----------------------------------+
